## Przygotowanie przed pierwszym uruchomieniem:
``` shell
./prepare.sh
```
## Włączenie programu przetwarzania:
``` shell
python main.py
```
## Włączenie analiz:
``` shell
cd analyses
python nazwa_analizy.py
```
